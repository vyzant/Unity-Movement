    Enum HumanBodyBonesMappings.BodyTrackingBoneId

    Body tracking bone IDs that should be exposed through the inspector. BoneId has enum values that map to the same integers, which would not work with a serialized field that expects unique integers. BodyTrackingBoneId is an enum that restricts BoneId to the values that we care about.
