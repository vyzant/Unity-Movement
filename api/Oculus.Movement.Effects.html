    CompareVectorHashes

    Generates a bunch of vertices that are close to each other. Compares how hash sets consolidate the two. One is based on Unity's generic Vector3 hash function while another is based on a fancier VertexKey structure. Unity's can be found here: https://github.com/Unity-Technologies/UnityCsReference/blob/master/Runtime/Export/Math/Vector3.cs#L229

    Custom editor class for comparing vector hashes.
